# Pokémon TCG Win/Loss Tracker

A personal win/loss tracker for Pokémon TCG matches, hosted via GitHub Pages.

## Features
- 🃏 **Deck Management** — add and manage your decks
- 📝 **Match Logging** — log wins/losses with opponent deck and event type
- 📊 **Stats Dashboard** — win rates by deck and event
- 📋 **Match History** — filterable log of all matches
- 💾 All data saved locally in your browser (localStorage)

## Event Types Supported
League · TCGLive · Challenge · Cup · Regional · International · Worlds

## Usage
Visit the live site: https://brammeketv.github.io/winlossdatapkmn

Data is stored in your browser. No account or server needed.
